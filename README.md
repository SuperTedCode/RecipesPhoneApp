# RecipesPhoneApp
Ionic application built for android displaying different recipes and giving user access to phones camera and gallery to add/edit recipes stored on the phone.
Build for android@5.1.1 this app was designed to use some functionality on the phone. This app uses the camera and the gallery. It can store and edit 
data on the phone. With the use of sliding menus and side-bars these functions can be accessed. Please have a look over the user guide.
